# personal-site

> 在线地址：**[personal-site-eight-rho.vercel.app](https://personal-site-eight-rho.vercel.app)**

个人展示网站，包含 4 个作品项目的介绍与下载入口。

## 技术栈

HTML + CSS + Vanilla JS，部署于 Vercel。

## 项目列表

| 项目 | 说明 | 类型 |
|------|------|------|
| 个人网站 | 极简风格响应式首页，深色/浅色主题切换 | Web |
| ClipboardHistory | Windows 剪贴板历史管理工具，WPF .NET | 桌面 |
| PlanClip | 计划清单 + 粘贴板历史二合一，开机弹出计划页 | 桌面 |
| 舒尔特表 | 专注力训练 PWA 游戏，Android APK 可安装 | 游戏 |
