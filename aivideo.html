AIvideo

全自动 AI 短视频流水线。一句话主题 → LLM 剧本 → AI 生视频 → TTS 配音 → FFmpeg 合成 → 一键发布抖音。三道坎压成一句话。

GitHub 开源
Python MCP Server Claude Code Edge TTS FFmpeg Playwright

项目概述

做 AI 短视频有三道坎：写剧本耗脑力、AI 生视频要来回调、发抖音得手动填描述标签。AIvideo 把三道坎压成一句话 — 你说主题，剩下全自动。基于 Claude Code MCP 协议深度集成，/makevideo 命令触发全套流水线。适合做连载故事的创作者、AI 内容矩阵和个人博主。

核心功能

主题 → 剧本

LLM 根据一句话主题自动生成分镜脚本 JSON + 元数据，故事连贯有节奏

多模型支持

默认 Agnes 免费 API（0 成本），可切换 Seedance / Kling 付费模型获更高画质

自动排队优化

MCP 秒返 task_id，Bash 后台轮询不阻塞工作流，完成后自动接续下一步

Edge TTS 配音

微软 Edge TTS 免费语音合成，自动匹配剧本旁白，无需额外 API Key

FFmpeg 音画合成

自动拼接视频片段 + 混合 TTS 音频 + 抖音规格转码，输出 final_douyin.mp4

一键发布抖音

Playwright 自动化填写标题、描述、标签、AI 声明，浏览器不关交给用户最后确认

Claude Code 集成

MCP Server 注册为 video-pipeline，/makevideo Skill 一句话即可启动全流程

Windows 友好

处理 GBK 编码兼容、asyncio 隔离等 Windows 特有坑，开箱即用

技术架构

MCP Server

FastMCP 框架，4 个工具：submit_video / poll_video / list_outputs / cleanup_outputs

monitor_task

后台轮询 Agnes 视频生成进度，完成后自动保存 video_results.json

editor

TTS 语音合成 + FFmpeg 片段拼接 + 音频混流 + 抖音 H.264 转码

publisher

Playwright Chromium 打开抖音创作者中心，自动填入标题/描述/标签/AI声明/定时发布
