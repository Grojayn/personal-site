舒尔特表

专注力训练 PWA 网页游戏。莫兰迪配色，支持 3 种尺寸 × 2 种模式，手机可安装到桌面全屏运行，离线可用。

开始游戏 下载 Android APK
HTML5 CSS3 JavaScript PWA Web Audio API

项目概述

一款舒尔特表专注力训练游戏。按数字顺序或倒序依次点击网格中的数字，锻炼视觉搜索速度和注意力集中度。PWA 架构支持手机端添加到桌面，全屏独立运行，离线可用。采用莫兰迪色系配色，界面安静舒适，减少视觉干扰。

游戏功能

3 种网格尺寸

4×4（入门）、5×5（标准）、8×8（挑战），适配不同训练强度

双模式

顺序 1→N 和倒序 N→1，切换训练策略避免惯性思维

自动计时

首次点击自动启动计时器，精确到 0.1 秒，完成即停

最佳记录

LocalStorage 持久化存储各模式最佳成绩，自动排序展示

触觉反馈

点对短暂振动 + 点错长振动，不用看也能感知操作结果

音效反馈

Web Audio API 合成音效，点对双音上升、点错低频、完成旋律祝贺

倒计时启动

3-2-1-开始 倒计时动画，数字隐藏期间防止提前记忆位置

莫兰迪随机配色

每格子随机从 10 种莫兰迪色中选取，视觉干扰低，每局配色不重复

PWA 特性

可安装

手机浏览器访问后添加到主屏幕，全屏独立运行，如同原生 App

离线可用

Service Worker 缓存全部静态资源，无网络也能正常打开训练

安全区适配

支持刘海屏安全区自动内边距适配，沉浸式全屏体验

Android APK

基于 TWA 打包为独立 APK，可直接安装到 Android 手机使用

技术架构

纯前端

单页 HTML + Vanilla JS + CSS，零依赖零框架，40KB 内即可完整运行

Web Audio API

OscillatorNode + GainNode 合成音效，无需加载音频文件

Service Worker

install 预缓存全静态资源，fetch 缓存优先策略保证离线可用

Web App Manifest

standalone 显示模式 + 竖屏锁定 + 自定义图标和主题色
