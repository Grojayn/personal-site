ClipboardHistory

轻量级 Windows 剪贴板历史管理工具，WPF .NET 6.0 构建，常驻系统托盘，Alt+V 秒速唤起。

C# 12 WPF .NET 6.0 Win32 API Windows

项目概述

一款常驻系统托盘的 Windows 剪贴板增强工具。实时监控剪贴板变化，自动记录文本和图片内容，通过全局热键 Alt+V 快速唤起浮窗，支持搜索、内联编辑、图钉固定、一键粘贴到任意应用。窗口可自由拖拽调整大小，底部一键清空全部历史。

核心功能

实时监控

Win32 AddClipboardFormatListener 监听剪贴板，文本与图片自动捕获

SHA256 去重

对连续相同内容自动去重，避免重复记录

全局热键

Alt+V 随时唤起浮窗，不打断当前工作流

一键粘贴

点击卡片自动回写剪贴板并模拟 Ctrl+V，粘贴到任意应用

内联编辑

点击卡片编辑按钮直接修改文本内容，Enter 保存 Esc 取消

搜索过滤

实时关键词搜索，快速定位历史记录

图钉固定

固定常用内容置顶显示，不受过期清理影响

7天自动清理

上限 100 条，非固定内容 7 天后自动过期，底部一键清空全部

自由窗口

标题栏拖拽移动，边缘 6px 热区拖拽调整大小，开机自启后台运行

技术架构

Models

C# record 定义 ClipboardItem，支持 Text/Image 两种类型，含 Id、时间戳、置顶状态

ClipboardMonitor

HWND 消息钩子 + PNG 编码 + SHA256 哈希对比，解耦事件通知

StorageService

SemaphoreSlim 线程安全 JSON 持久化，支持增删改查、置顶切换、内联编辑、过期清理

PasteService

Clipboard.SetText/SetImage 回写 + SendInput 模拟 Ctrl+V 四步按键序列，异步执行
