PlanClip

计划清单 + 剪贴板历史二合一桌面工具。开机弹出计划页，Alt+V 秒速唤起，计划与粘贴板自由切换。

C# 10 WPF .NET 6.0 Win32 API Windows

项目概述

在 ClipboardHistory 基础上全面升级的二合一效率工具。保留了粘贴板历史全部功能的同时，新增完整的计划清单（Todo List）模块。开机自动弹出计划页，帮助每天第一时间聚焦当天任务。计划与粘贴板两个模块通过顶部按钮一键切换，共用搜索框和全局热键，交互自然流畅。

计划清单功能

快速添加

底部 [+ Add Plan] 按钮或搜索框输入回车一键添加待办事项

截止日期+时间

DatePicker 选择日期 + 独立小时/分钟输入框，默认 23:59，自动按日期排序

分类筛选

顶部 [All ▼] 下拉 Popup，一键切换 All / Pending / Completed 视图

内联编辑

点击 ✏️ 按钮直接修改计划标题，Enter 保存 Esc 取消

完成标记

✅ 标记完成，按钮变绿 #4CAF50，自动归入 Completed 分类

置顶排序

📌 置顶/取消置顶，置顶项优先排列；排序规则：置顶 > 截止日期 > 创建时间

清空计划

底部 [Clear All] 一键清除所有计划项

粘贴板历史功能

实时监听

Win32 AddClipboardFormatListener 监听剪贴板，文本与图片自动捕获，延迟 < 500ms

SHA256 去重

连续相同内容自动去重，避免重复记录

一键粘贴

点击卡片回写剪贴板 + 模拟 Ctrl+V，直接粘贴到任意应用

图钉固定 + 7天清理

上限 100 条，非固定内容 7 天后自动过期，底栏一键清空全部

通用功能

功能切换

顶部 [Plan] / [Clipboard] 按钮自由切换，选中态蓝色高亮

统一搜索

顶部搜索框同时对计划标题和粘贴板文本进行实时关键词过滤

全局热键

Alt+V 随时随地唤起/隐藏浮窗，不打断当前工作流

开机自启 + 系统托盘

随 Windows 开机启动，默认显示计划页；托盘图标右键 Exit 退出

独立持久化

计划数据存为 plans.json，粘贴板数据存为 clipboard.json，互不干扰

自由窗口

320px 宽可拖拽窗口，适应更多内容展示，支持边缘拖拽调整大小

技术架构

Models

PlanItem record（Id、Title、DueDate、Status、IsPinned）+ ClipboardItem record，PlanStatus 枚举 Pending/Completed/Cancelled

PlanStorageService

SemaphoreSlim 线程安全 JSON 持久化，支持增删改查、完成标记、置顶切换、截止日期管理

MainWindow

双 ItemsControl 按 Visibility 切换页面，_currentPage + _planFilter 状态管理，统一搜索过滤逻辑

复用模块

ClipboardMonitor、PasteService、StorageService、AutoStartService 从项目一继承，保持一致

下载使用

下载 PlanClip.exe

自包含单文件，无需安装运行时。下载后双击运行，自动缩小到系统托盘，按 Alt+V 开始使用。开机默认显示计划清单页。
